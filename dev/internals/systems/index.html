PNSystem{T, PNOrder}

Base type for all PN systems, such as BBH, BHNS, and NSNS.

These objects encode all essential properties of the binary, including its current state. As such, they can be used as inputs to the various fundamental and derived variables, as well as PN expressions and dynamics functions.

All subtypes should contain a state vector holding all of the fundamental variables for the given type of system. The parameter T is the type of the state vector â€” for example, Vector{Float64}. PNOrder is a Rational giving the order to which PN expansions should be carried.
